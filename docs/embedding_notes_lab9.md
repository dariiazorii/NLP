# Embedding Notes: Lab 9

### 1. Опис корпусу
* **Джерело:** Оголошення та документація із системи ProZorro.
* **Тематика:** Технічне обслуговування, юридичні послуги (БВПД), постачання енергоресурсів.
* **Стан даних:** Попередньо очищений текст (`processed_v2`), збережений у вихідній морфологічній формі (без лематизації).

### 2. Моделі
Було натреновано та порівняно дві архітектури:
1.  **Word2Vec:** Класична модель, що базується на цілих словах.
2.  **FastText:** Модель, що враховує субсимвольні n-грами (ефективна для української морфології).

### 3. Параметри навчання (Hyperparameters)
Для обох моделей використано ідентичний набір параметрів:
* `vector_size`: 100
* `window`: 5
* `min_count`: 3
* `sg`: 1 (Skip-gram)
* `workers`: 4
* `seed`: 42

### 4. 10 слів для аналізу (Nearest Neighbors)
Для загального аналізу обрано:
*закупівля, грн, адвокат, електроенергія, бвпд, договір, прозоро, термін, ліфтів, вимоги.*

### 5. 5 доменних термінів
Детальний розбір проведено для:
1. **електроенергія** (енергетика)
2. **адвокат** (юридичні послуги)
3. **договір** (документообіг)
4. **лот** (структура тендера)
5. **закупівля** (процедура)

### 6. 5 кейсів “корисно / некорисно”
* **Кейс 1: Електроенергія (Корисно).** FastText успішно зібрав відмінки слова.
* **Кейс 2: Договір (Корисно).** Обидві моделі знайшли релевантний контекст укладання.
* **Кейс 3: Адвокат (Некорисно).** Word2Vec видав OOV, FastText додав семантичний шум.
* **Кейс 4: Лот (Некорисно).** Обидві моделі надто звузили контекст до інтернет-послуг.
* **Кейс 5: Закупівля (Частково корисно).** Сусіди логічні, але занадто шаблоні/канцелярські.

### 7. Порівняння Word2Vec vs FastText
| Критерій | Word2Vec | FastText |
| :--- | :--- | :--- |
| **Морфологія** | Слабко (сприймає відмінки як різні слова) | **Відмінно** (групує через n-грами) |
| **Рідкісні слова** | Часто видає OOV | Побудує вектор за частинами слова |
| **Чистота контексту** | Висока на дуже частих словах | Може містити шум через схожі суфікси |
| **Вердикт** | Простіший, але менш ефективний | **Кращий для української мови** |
