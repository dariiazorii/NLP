### 1. Коди класифікатора CPV (Єдиний закупівельний словник)
* **Визначення:** Восьмизначний цифровий код формату `XXXXXXXX-Y` (де Y — контрольна цифра), що офіційно класифікує предмет закупівлі згідно з ДК 021:2015.
* **Формат на виході (нормалізація):** Рядок у стандарті `XXXXXXX-Y` (наприклад, `44110000-4`). Видаляються префікси "ДК 021:2015", зайві пробіли та крапки.
* **Що НЕ вважаємо полем:**
* Довільні набори цифр (наприклад, номер тендера або ЄДРПОУ `34567890`).
* Коди попередніх класифікаторів (наприклад, ДК 016:2010), якщо вони не відповідають структурі CPV.
* **5 Edge Cases з датасету:**
1. Код без контрольної цифри (наприклад, `44110000`).
2. Код, записаний через пробіл замість дефіса (`44110000 4`).
3. Група кодів, перелічених через кому (`44110000-4, 44111000-1`).
4. Код всередині дужок: `(за ДК 021:2015: 44110000-4)`.
5. Помилкове написання префікса: `ДK 021-2015` (з латинською "K").

### 2. Грошові суми (Очікувана вартість)
* **Визначення:** Числове значення вартості закупівлі разом із валютою, виражене цифрами (може містити розділювачі тисяч та копійки).
* **Формат на виході (нормалізація):** Float значення (число з рухомою комою) + ISO код валюти. Приклад: `500000.00 UAH`.
* **Що НЕ вважаємо полем:**
* Відсотки (наприклад, "ПДВ 20%").
* Кількість товару, що схожа на суму ("1000 шт").
* Дати або номери законів.
* **5 Edge Cases з датасету:**
1. Сума з пробілом як розділювачем тисяч: `150 000,00 грн`.
2. Валюта записана скорочено без крапки або з нею: `грн.`, `грн`, `гр`.
3. Суми в іноземній валюті: `1200.50 USD` або `€ 500`.
4. Запис суми без копійок: `10000 UAH`.
5. Сума, де замість коми використано крапку для копійок: `450.75 грн`.

### 3. Одиниці виміру та Кількість
* **Визначення:** Кількісний показник предмета закупівлі, що складається з числа та стандартного скорочення одиниці виміру (метри, кілограми, штуки тощо).
* **Формат на виході (нормалізація):** JSON-об'єкт `{"quantity": float, "unit": string}`. Приклад: `{"quantity": 10.5, "unit": "kg"}`.
* **Що НЕ вважаємо полем:**
* Порядкові номери ("Лот №1").
* Технічні характеристики без кількості ("товщина 5 мм").
* Математичні формули в описі.
* **5 Edge Cases з датасету:**
1. Скорочення "штук" у різних варіантах: `шт.`, `од.`, `компл.`.
2. Дробові значення через дефіс або пробіл: `0,5 кг` або `1/2 т`.
3. Складні одиниці виміру: `людино-години`, `маш-год`, `упак (10 шт)`.
4. Кількість, записана словами (якщо зустрічається): `десять метрів` (хоча ми фокусуємось на цифрах).
5. Відсутність пробілу між числом та одиницею: `100кг`, `50шт`.
