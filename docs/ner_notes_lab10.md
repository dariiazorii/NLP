# NER Notes: Lab 10 

## 1. Обраний Pipeline
Для роботи обрано **uk_core_news_trf** (spaCy). Це Transformer-модель на базі RoBERTa, яка є найточнішим інструментом для української мови завдяки глибокому розумінню контексту та морфології.

## 2. Сутності "з коробки"
Модель стандартно підтримує схему OntoNotes, зокрема:
* **PERSON:** Прізвища, імена, по батькові.
* **ORG:** Компанії, установи, державні органи.
* **GPE:** Міста, країни, регіони.
* **LOC:** Географічні об'єкти, вулиці, адреси.
* **PRODUCT, EVENT, FAC, NORP.**

## 3. Важливі доменні сутності
Для корпусу тендерної документації пріоритетними є:
* **DATE:** Терміни виконання та дати підписання.
* **MONEY:** Суми контрактів та ПДВ.
* **CPV:** Коди класифікатора (напр., 50750000-7).
* **ID:** Номери доручень, договорів та наказів.

## 4. Додані правила (Hybrid Rules)
Через компонент `EntityRuler` додано:
* **Словникові патерни:** Для текстових дат (назви місяців) та доменних абревіатур (НСЗУ, БВПД).
* **Логічні правила токенів:** Використання `IS_DIGIT`, `LIKE_NUM` та `PREFIX` для виявлення сум та номерів документів без використання складних регулярних виразів (для стабільності в Python 3.12).

## 5. Результати після впровадження правил
* **Recall (Повнота):** Радикально зросла для дат та кодів CPV, які статистична модель раніше ігнорувала.
* **Boundary Fixes:** Номери будинків тепер стабільно включаються до складу адрес (`LOC`), а валюта "грн" — до грошових сум.

## 6. Залишкові помилки
* **False Positives:** Через пріоритет правил іноді звичайні слова маркуються як `ID` або `ORG`.
* **Type Errors:** Модель все ще може плутати `GPE` та `LOC` для назв населених пунктів.
* **Segmentation:** Складні переліки адрес або сум іноді розбиваються на дрібні сутності.

## 7. План подальших вдосконалень
* **Звуження патернів:** Додавання контекстних обмежень (напр., шукати суму тільки після слова "вартість").
* **Пост-процесинг:** Об'єднання розірваних токенів (merging spans) після завершення роботи NER.
* **Fine-tuning:** Донавчання Transformer-шару на розміченому вручну тендерному датасеті для кращої адаптації до офіційно-ділового стилю.
