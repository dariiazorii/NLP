1. Задача та текстове поле
Задача: C (Пошук/Retrieval).
Text field: Для обробки обрано поле full_content, яке є конкатенацією назви тендера (title) та його детального опису (description). Це забезпечує повний контекст для пошукового індексу.

2. Як запустити ноутбук у Colab
Відкрийте labs/lab02/lab2_cleaning_normalization.ipynb у Google Colab.
Натисніть Runtime -> Run all. Ноутбук автоматично встановить залежності, завантажить raw.csv за прямим посиланням з Drive та виконає препроцесинг.

3. Правила очищення та нормалізації
Очищення: Видалення HTML-тегів, зайвих пробілів, невидимих символів та рядків, що не містять літер (суто цифри або спецсимволи).
Нормалізація: Зведення до нижнього регістру, уніфікація символів (тире, лапки, апострофи).
Виправлення гомогліфів: Автоматична заміна латинських літер, схожих на кириличні (наприклад, "i", "a", "o", "p"), що виникають через помилки розкладки.
Маскування PII: Заміна конфіденційних даних (номери телефонів, email, URL) на теги-заповнювачі: <PHONE>, <EMAIL>, <URL>.

4. 5 найболючіших edge cases
Мішана розкладка (Homoglyphs): Слова, де частина літер латинські, які не знаходяться звичайним пошуком.
Технічний шум: Тендери, опис яких складається лише з артикулів або ГОСТів (наприклад, "123-45/67"), що не несуть семантичного змісту.
Контактні дані в тексті: Особисті номери телефонів та пошти директорів, вказані прямо в описі предмета закупівлі.
Короткі тексти: Записи типу "Див. додаток" або "Тендер", які є неінформативними для моделі та видаляються як is_valid=False.
Розриви скорочень: Крапки після "грн.", "вул.", "м.", які можуть помилково сприйматися як кінець речення при токенізації.
5. Результати "після"
Очищення від шуму: Видалено {кількість} невалідних записів (короткі тексти та "сміття"), що зменшило обсяг даних на ~5%, залишивши лише якісний контент.
Безпека даних: Успішно замасковано понад {кількість} випадків PII (телефонів та пошт), забезпечуючи анонімність даних.
Стабільність: Досягнуто 100% ідемпотентності пайплайну — повторна обробка вже очищеного тексту не змінює результат.
