# Лабораторна робота №9:  Word Embeddings (Word2Vec / FastText)

### 1. Використаний корпус
Для навчання використано очищений корпус оголошень **ProZorro** (`processed_v2`). 
* **Особливість:** Тексти не проходили етап лематизації. Це дозволило перевірити стійкість моделей до багатого морфологічного складу української мови.
* **Домен:** Державні закупівлі (енергетика, юриспруденція, технічне обслуговування).

### 2. Натреновані моделі
Було реалізовано дві основні архітектури:
1. **Word2Vec (Gensim):** Модель, що навчається на цілих токенах (словах).
2. **FastText (Gensim):** Модель, що використовує субсимвольні n-грами, що дозволяє працювати зі словами, яких немає у словнику.

### 3. Параметри навчання
Обидві моделі використовують ідентичні гіперпараметри для забезпечення валідності порівняння:
* `vector_size`: 100 (розмірність вектора)
* `window`: 5 (контекстне вікно)
* `min_count`: 3 (мінімальна частота слова)
* `sg`: 1 (використано архітектуру Skip-gram)
* `seed`: 42 (для відтворюваності результатів)

### 4. Типи аналізованих слів
Для оцінки якості моделей було відібрано 10 слів різних категорій:
* **Часті загальні слова:** *закупівля, вимоги*.
* **Доменні терміни:** *електроенергія, адвокат, бвпд*.
* **Абревіатури та специфічні токени:** *грн, прозоро*.
* **Слова з морфологічною варіативністю:** *договір, ліфтів*.

### 5. 5 найцікавіших кейсів аналізу
1. **Електроенергія:** FastText ідеально згрупував відмінки, тоді як Word2Vec видав технічний шум.
2. **Договір:** Word2Vec показав чудові функціональні зв'язки (укладення, переможець).
3. **Адвокат:** Кейс із OOV (Out-of-Vocabulary), де Word2Vec не зміг знайти слово, а FastText виявив зв'язок із правовою допомогою.
4. **Лот:** Приклад зміщення вектора в сторону вузької тематики інтернет-провайдерів через специфіку датасету.
5. **Закупівля:** Приклад стабільного семантичного сусідства для частих слів в обох моделях.

### 6. Порівняння Word2Vec vs FastText
* **Де FastText був кращим:** У роботі з нелематизованими текстами. Він успішно об'єднує різні форми слова (`договору`, `договором`) та працює з рідкісними термінами.
* **Де виграшу майже не було:** На дуже коротких словах або абревіатурах, а також у випадках, коли Word2Vec видавав більш точні "смислові" (дієслівні) зв'язки без морфологічного шуму.

### 7. Висновок щодо корисності
Embeddings є **реально корисними** для даного корпусу. Вони дозволяють:
* Знаходити контекстуальні синоніми, які неможливо виявити частотними методами.
* Автоматично групувати терміни за тематиками (енергетика, юриспруденція).
* Покращити якість майбутніх моделей класифікації за рахунок більш щільного та змістовного представлення тексту.
